# LG 6.0 & 6.1 第二级

在 CSS 文件中写入你在讲义上写的规则。

## 你需要做什么

1. 打开 `style.css` 文件
2. 写入你在讲义上完成的 CSS 规则
3. 在浏览器中打开 `index.html` 检查你的作品
4. 如果和讲义上的模型图一样，你就完成了！

## 文件夹里的文件

- `index.html` — HTML 页面，不要编辑这个文件
- `style.css` — 在这里写你的 CSS 规则

## 提醒

- 类选择器以 `.` 开头
- ID 选择器以 `#` 开头
- CSS 规则的格式如下：

```css
选择器 {
  属性: 值;
}
```
